# 🎤 interview-me

> 최근 작성한 코드로 기술 면접 연습하기

당신이 작성하거나 수정한 코드를 시니어 엔지니어 관점에서 분석하고, 실제 면접에서 나올 법한 꼬리질문을 생성합니다.

## 설치

```bash
# 1. 마켓플레이스 등록
/plugin marketplace add mun-jeong-min/interview_plugin

# 2. 플러그인 설치
/plugin install interview-me@interview-me-marketplace
```

## 사용법

```bash
# 기본: 최근 커밋의 변경사항 분석 (질문 3개)
/interview-me

# 질문 개수 조정 (최대 7개)
/interview-me --count=5

# 영어로 질문
/interview-me --lang=en
```

## 옵션

| 옵션 | 설명 | 기본값 |
|------|------|--------|
| `--count=<n>` | 질문 개수 (1~7) | 3 |
| `--lang=<ko\|en>` | 출력 언어 | ko |

## 질문 카테고리

| 카테고리 | 이모지 | 주요 주제 |
|----------|--------|-----------|
| Security | 🔒 | 인증, 인가, 인젝션, 암호화 |
| Concurrency | ⚡ | 레이스 컨디션, 트랜잭션, 락 |
| Performance | 🚀 | N+1, 캐싱, 인덱싱, 메모리 |
| Design | 🏛 | SOLID, 패턴, 결합도, 추상화 |
| Testing | 🧪 | 커버리지, 모킹, 엣지케이스 |
| Observability | 👁 | 로깅, 메트릭, 트레이싱 |
| Data | 🗄 | 스키마, 마이그레이션, 일관성 |

## 난이도

- **★☆☆ (기본기)**: 모든 개발자가 알아야 할 기초
- **★★☆ (실무)**: 실제 경험이 필요한 실무 시나리오
- **★★★ (심화)**: 깊은 전문성과 아키텍처 사고 필요
